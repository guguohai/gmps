# Admin Web Frontend

This is the management background frontend for the PS system.

## Project Status
Currently, the main development is happening in the `v3` directory at the root.

## Setup
To start development, refer to the `v3` directory:
- `npm install`
- `npm run dev`

## Future Migration
Plans are in place to migrate the contents of `v3` into this directory to maintain a mono-repo structure.
